# MeetSphere

MeetSphere is a modern, real-time video conferencing web application built to facilitate seamless online communication. It allows multiple users to join video calls, chat in real-time, manage calls, and track meeting history. With a sleek and responsive UI/UX, MeetSphere ensures a smooth and interactive meeting experience.

## Project Intro
MeetSphere is designed to provide a secure and intuitive platform for online meetings. Users can authenticate, join lobbies, share video and audio, chat in real-time, and manage their meetings efficiently. Each meeting generates a unique room URL that can be shared with participants for easy access.

## Getting Started

### Prerequisites
- Node.js and npm installed
- MongoDB database (local or cloud)
- Basic knowledge of React and Node.js

### Installation
1. Clone the repository:
   git clone https://github.com/Kishor-Pashte/MeetSphere

2. Navigate to the backend folder and install dependencies:
   cd meetsphere/backend
   npm install

3. Navigate to the frontend folder and install dependencies:
   cd ../frontend
   npm install

4. Running the Project

  - **Start the backend server:**
      cd backend
      npm run dev

  - **Start the frontend development server:**
      cd frontend
      npm start

Open http://localhost:3000 in your browser.

## Usage
   - Register or login to the platform.
   - Enter a username to join the lobby.
   - Enter a meeting code and click Connect to join a video conference room.
   - Use video, mic, screen sharing, and chat features as needed.
   - Share your unique room URL to invite others.
   - Track your meeting history, delete calls if needed, and safely logout.
